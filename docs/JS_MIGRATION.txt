PHP2Go Web Development Framework - JS MIGRATION
================================================

	Version 0.5.8 - October 2007
	---------------------------------
	http://www.php2go.com.br

	Below you can view a complete list of functions that used to be
	included in the PHP2Go distributions until version 0.4.3.

	The functions that have a "*" mark in the right side were included
	in a backwards compatibility library called compat.js. This library
	is included by default in every page built with php2go.base.Document.

	The other functions (those without the "*" mark) were dropped and will
	not be supported anymore. To keep these old functions running for your
	application, you'll have to copy the old code to your application's
	source tree.
